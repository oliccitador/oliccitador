# Leitor de Editais - Sprint 3

Modulo de analise automatica de editais com 9 agentes especializados.

## Funcionalidades

- Pipeline de 9 modulos
- 9 Agentes especializados
- CNPJ + Contexto operacional
- Perguntas PRE/POST analise
- QuestionRouter (11 categorias)
- Evidencias rastreaveis

## Instalacao

```bash
cd modules/leitor-editais
npm install
npx prisma generate
```

## Configuracao

Copie .env.example para .env.local e configure as variaveis.

## Uso

```bash
npm run dev
```
